🧮 Calculadora Simples em Java

Uma calculadora simples desenvolvida em Java utilizando Java Swing para a interface gráfica.
O projeto utiliza componentes como JFrame e JPanel para construir uma interface intuitiva e funcional.

✨ Demonstração

Interface gráfica criada com Swing, permitindo realizar operações matemáticas básicas de forma rápida e simples.


🚀 Funcionalidades

✔️ Interface gráfica com Java Swing
✔️ Operações matemáticas básicas
✔️ Estrutura simples e fácil de entender
✔️ Ideal para iniciantes em Java GUI

Operações disponíveis:

➕ Adição

➖ Subtração

✖️ Multiplicação

➗ Divisão
